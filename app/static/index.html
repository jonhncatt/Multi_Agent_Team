多 Role 图例 
阶段 3 状态: MVP 已完成

固定角色:
Router -> Coordinator -> Worker

按需角色:
Router -> Coordinator -> [Planner?] -> [Researcher/FileReader/Summarizer?] -> Worker -> [Conflict Detector?] -> [Reviewer?] -> [Revision?] -> [Structurer?]

术语:
- role = 所有执行角色的总称
- agent = 由 LLM 驱动的 role
- processor = 不依赖 LLM 的 role
- Router = agent + processor（规则路由 + 可选 LLM 路由）
- Coordinator = processor，负责切换 tool_mode、重绑 Worker、抑制反复确认
- Worker 一定存在
- worker_mode=direct_answer: Worker 直接回答
- worker_mode=uses_tools: Worker 会调用工具链

常见路径:
1. 简单理解
   Router -> Coordinator -> Summarizer -> Worker

2. 联网查证
   Router -> Coordinator -> Researcher -> Planner -> Worker(tools) -> Reviewer -> Revision? -> Structurer

3. 文档定位
   Router -> Coordinator -> FileReader -> Planner -> Worker(tools) -> Reviewer -> Structurer
